# alviny-task-8

alviny's 7th INDA task.

## How to run

Using RARS:

1. Clone the repository with

```bash
git clone https://github.com/IndaPlus25/alviny-task-8.git
```

2. Launch RARS.
3. Navigate to the program you would like to run by selecting File > Open... > \[your target directory here\] > alviny-task-8.
4. Continue navigating through the directory tree until you've reached the program you would like to run. There are two programs in this repo: `alviny-task-8/high-to-low/multiplication.asm` and `alviny-task-8/sieve/sieve.asm`.
